# Weather Now Dashboard

Weather Now is a modern weather application built with HTML, CSS, and JavaScript.
It allows users to search for any city and view real-time weather conditions, hourly forecasts, and a 7-day forecast through a clean and responsive interface.

## Features

* Real-time weather search by city
* Current temperature and weather conditions
* Hourly forecast with weather icons
* 7-day weather forecast
* Unit conversion for temperature, wind speed, and precipitation
* Search history and suggestions
* Loading states and error handling
* Responsive dashboard design

## Built With

* HTML5
* CSS3
* Vanilla JavaScript
* Open-Meteo API(Provides weather and forecast data)
* OpenStreetMap Nominatim API(Converts city names into coordinates)

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Search for a city to view its weather information.
